Search Endpoint

GET /api/search

Search for K-dramas across multiple sources.

Parameters:

  • q (required) - Search query (minimum 2 characters)
  • page (optional) - Page number (default: 1)
  • source (optional) - Source to scrape (all, dramacool, kissasian, default: all)
